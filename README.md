## 🚀 Usage

Follow the steps below to set up the toolchain and build the kernel.

### 1. Install the Toolchain

Run the following script to install the compatible toolchain required for this kernel source:

```bash
./toolchains.sh
```

### 2. Build the Kernel

After the toolchain has been installed, run:

```bash
./build.sh
```

This script will:
- Set up the required build environment.
- Compile the kernel using the installed toolchain.
